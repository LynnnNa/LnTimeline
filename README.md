# LnTimeline

+ 原生JS写的小程序,数据用fs存储在本地文件中
+ 用NW.js封装打包使用
+ 主页:[http://timeline.lynnote.site/](http://timeline.lynnote.site/)

##主要功能
+ 计时器.主界面以时间轴的形式展示
+ 计时设置.设置界面提供计时自定义功能

# LnTimeline

+ 用NW.js构建的桌面应用
+ 原生JS开发, 数据用fs存储在本地文件
+ 制作初衷是为了实现一个时间轴计时应用,"0.2.120190321"之后的版本,有部分功能需要读取游戏客户端本地数据,主要用于提高<剑网三>的游戏体验,该部分功能由用户手动授权



+ 主页:[http://timeline.lynnote.site/](http://timeline.lynnote.site/)

## 主要功能
+ 计时器.主界面以时间轴+列表的形式展示
+ 设置.自定义时间轴,快捷键等
+ 生活技能.读取游戏插件(AH插件)本地数据, 根据AH浏览的最新商品记录计算相关产出成本
